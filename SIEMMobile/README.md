# SIEM Mobile

Expo React Native app for the SIEM frontend transition.

## Run

```bash
cd SIEMMobile
npm install
npx expo start
```

## API URL

Edit `src/services/api.js` when switching devices.

- Android emulator: `http://10.0.2.2:5000/api`
- Physical phone: `http://YOUR_COMPUTER_LAN_IP:5000/api`
